# Drogaria Max Cloud - SaaS para Farmácias

## Setup
1. Instale as dependências: `npm install`
2. Configure as chaves no `.env` (Gemini API Key)
3. Inicie o servidor: `npm run dev`

## Estrutura
- `app/`: Rotas e Páginas
- `components/`: Componentes Reusáveis
- `lib/`: Utilitários e IA
- `hooks/`: Custom Hooks para estado
- `public/`: Assets estáticos

## Funcionalidades
- Dashboard Premium
- CRM com Kanban
- Chat Omnichannel (Simulado)
- IA Nanda integrada para vendas e suporte
- Automação de Marketing
